Example Project

The teacher assigned this project so we can better understand and get used to the command line. He also assigned it so we can work on a single github repo with a group of people. It helped us practice git pulling/pushing, merging conflicts; git add., git commit -m "",etc. The difficulty of the project was that it was hard and confusing to organize everything and it was hard to get everyone to work on it the same day as someone may be busy. The next time I do something similar to this project I would communicate better with my teammates and check on every error or mistake I've made.
